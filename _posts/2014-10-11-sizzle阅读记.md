最近想看点儿框架之内的代码充实一下，于是选择了sizzle这个框架，很喜欢jQuery的风格，一有时间就记记。

### faster indexOf

```javascript
// #line 61
var indexOf = function( list, elem ) {
	var i = 0,
	  len = list.length;
	for ( ; i < len; i++ ) {
		if ( list[i] === elem ) {
			return i;
		}
	}
	return -1;
}
```
此方法的效率比原生数组的indexOf效率更高
