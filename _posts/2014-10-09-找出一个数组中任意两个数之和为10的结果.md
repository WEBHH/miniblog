## 找出一个数组中任意两个数之和为10的结果

```
input
 arr 一个整数数组
output
 所有相加和为 10 的结果
egg:
input [1, 2, 3, 4 ,5, 6, 7, 8, 9] output [[1,9],[2,8],[3,7],[4,6]]
```

## 我的解法

```javascript
function(arr) {
    var tmp = {};
    var len = arr.length;
    var result = [];
    for(var i=0; i<len; i++) {
        tmp[arr[i]] ? tmp[arr[i]]++ : (tmp[arr[i]] = 1);
    }
    for(var k in tmp) {
        while(tmp[k]) {
            tmp[k]--;
            if(tmp[10-k]) {
                tmp[10-k]--;
                result.push([k, 10-k]);
            }
        }
    }
    return result;
}
```
